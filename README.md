# To-Do App - Server and Client Setup Guide

## Server Setup

1. ** Goto Server **
   ```bash
   cd todo-server
   ```
2. ** Configure Environment Variables **

   ```bash
   cp .env.example .env
   ```

   Edit .env file if necessary to set up environment variables

3. ** Install Dependencies **
   ```bash
   npm install
   ```
4. ** Start the Server **
   ```bash
   npm start
   ```
